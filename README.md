# 🏔️ ALTIGUIDE
**Aplikasi Navigasi Spasial Offline & Pusat Informasi Pendakian Jawa Tengah**

ALTIGUIDE dirancang untuk membantu pendaki gunung di Jawa Tengah tetap aman dengan fitur navigasi tanpa koneksi internet dan informasi basecamp yang terintegrasi.

## 📌 Fitur Utama
* **Peta Pendakian Offline:** Navigasi jalur pendakian tanpa sinyal seluler.
* **Informasi Basecamp:** Data lengkap kontak dan fasilitas basecamp.
* **Prediksi Cuaca:** Informasi cuaca terkini di area pegunungan.
* **Dashboard Admin:** Manajemen data pendakian dan status jalur.

## 📂 Struktur Repositori
* `src/`: Kode sumber aplikasi (Frontend & Backend).
* `docs/`: Dokumentasi teknis, kontrak tim, dan laporan praktikum.
* `tests/`: Dokumentasi pengujian sistem (QA).
